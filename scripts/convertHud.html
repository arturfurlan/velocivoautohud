Converter HUD para PNG transparente

Este script converte a imagem HUD.jpg em um PNG transparente.

Imagem Original

Resultado (fundo transparente)

Como usar:

  1. Clique em "Carregar HUD.jpg" para carregar a imagem.
  2. Ajuste a cor que deseja tornar transparente usando o seletor de cores.
  3. Ajuste a tolerância conforme necessário para pegar diferentes tons da cor.
  4. Clique em "Baixar PNG Transparente" para salvar o resultado.
  5. Coloque o arquivo PNG baixado na pasta public/ do projeto.
